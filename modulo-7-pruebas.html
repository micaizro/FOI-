Fundamentos de Instrucción

MÓDULO 5: PREPARACIÓN DE INSTALACIONES Y EQUIPOS

INTRODUCCIÓN

La preparación adecuada del aula, áreas de estudio y equipos es fundamental para garantizar que la instrucción se desarrolle de manera fluida y efectiva.

Estos preparativos deben realizarse antes de cada sesión, asegurando que tanto el instructor como los participantes puedan concentrarse en el aprendizaje sin interrupciones.

OBJETIVO

Preparar correctamente el entorno de aprendizaje (instalaciones y equipos) para facilitar una instrucción clara, organizada y efectiva.

1. ARREGLO DEL AULA / ÁREAS DE ESTUDIO

El aula puede organizarse de diferentes formas según las necesidades de aprendizaje.

No existe una única forma correcta, pero sí principios básicos que deben cumplirse.

2. PREPARATIVOS EN EL AULA

Para lograr un entorno adecuado, el instructor debe considerar:

🔹 Espacio de trabajo

  • Proporcionar suficiente espacio para que los alumnos trabajen cómodamente.

 
🔹 Visión

  • Asegurar que todos los alumnos puedan ver claramente el material proyectado.

 
🔹 Montaje de equipos

  • Cubrir toda la pantalla
  • No bloquear la visión de ningún estudiante

 
🔹 Distracciones

  • Eliminar elementos que interfieran con la atención del alumno

 
🔹 Disposición del aula

  • Maximizar superficie de trabajo
  • Asegurar buen ángulo de visión

3. MEDIOS DE EXPOSICIÓN

Los métodos de enseñanza deben adaptarse al tipo de contenido.

 
🔹 Presentación visual
Se usa cuando:

  • El tema involucra objetos concretos observables

 
🔹 Uso del color
Se recomienda cuando:

  • Se requiere distinguir elementos
  • Se explican diagramas complejos

 
🔹 Representación del movimiento
Se utiliza cuando:

  • Se deben aprender movimientos
  • Se explican partes móviles

 
🔹 Modelos
Son útiles cuando:

  • El objeto es tridimensional
  • Es difícil comprenderlo en 2D

 
🔹 Palabras escritas
Se usan cuando:

  • El contenido es complejo o crítico

 
🔹 Palabras habladas
Se usan cuando:

  • El contenido es simple
  • Se necesita motivar

4. USO DE PROYECTORES DE DIAPOSITIVAS

Características:

  • Requiere ambiente oscuro
  • Fácil de transportar
  • Bajo costo
  • Permite modificar contenido fácilmente

5. RECOMENDACIONES DE USO

  • Iniciar y finalizar con diapositiva oscura
  • Llenar la pantalla
  • Evitar luz que afecte la proyección
  • Mantener presentaciones cortas
  • Usar formato horizontal
